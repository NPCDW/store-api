## 简介

`store-api` 的一个 `springboot` 版本，因内存占用太大，目前已用 `Vert.x` 重构，可以参见
[https://github.com/NPCDW/store-api-vertx](https://github.com/NPCDW/store-api-vertx)

如果不关注内存占用的话，那我只推荐你 `springboot`，生态全，文档多，教程多，占据绝大多数的市场。
